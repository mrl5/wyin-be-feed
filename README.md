## Getting started
1. [Download and install Docker]

2. Build a container
```
make build
```
or
```
docker build -t wyin-be-feed .
```

3. Run server
```
make run
```
or
```
docker run -p 8080:8080 wyin-be-feed:latest
```

4. Query `/health` endpoint
```
curl -i http://localhost:8080/health
```
